# SECURITY

## Reporting a Vulnerability

If you discover a security vulnerability within ZarPresence, please do not open a public issue. Instead, please report it via GitHub's private vulnerability reporting or contact the maintainers directly.

We take security seriously and will investigate all reports as quickly as possible.

### Supported Versions

Only the latest version of ZarPresence is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |

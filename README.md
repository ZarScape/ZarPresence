# ZarPresence

Modern Discord Rich Presence, redefined

### Currently Supported
- YouTube & YouTube Shorts
- Crunchyroll
*(More platforms coming soon!)*

## Features
- **Privacy Mode**: Hide video titles and thumbnails on Discord.
- **Auto-Sync**: Real-time presence updates via WebSocket.
- **Dashboard**: Modern UI to manage platforms and settings.
- **Startup**: Option to start with Windows.

## Installation

### Prerequisites
- Node.js
- Rust (for Tauri)

### Setup
```bash
npm install
npm run build:extension  # Build the browser extension
npm run tauri dev        # Run the desktop app
```

## Extension
Load the `dist-extension` folder as an unpacked extension in your browser's developer mode.
